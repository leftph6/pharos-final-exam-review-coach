## Project Name

Final Exam Review Coach

## One-Liner

A reusable education Skill that turns messy course materials into exam portraits, operator-level notes, standard answers, weak-point diagnosis, and targeted reinforcement packs.

## Short Description

Final Exam Review Coach lets a Pharos agent become an exam-prep specialist. It reads a course corpus, identifies likely exam question slots, compresses knowledge into callable operators, answers questions in exam-standard form, diagnoses failed student attempts, and generates the next drills or mock paper. The Skill is designed for Phase 2 composition with payment, credential, or bounty Skills on Pharos.

## Long Description

This Skill was distilled from a real multi-subject final-exam workspace spanning DSP, financial mathematics, PDE numerical methods, data structures, numerical analysis, probability/statistics, optimal control, Japanese, accounting, finance, and economics.

The core idea is operator-level tutoring. Instead of only giving a final answer, the Skill names the reusable move a student needs: question type, operator chain, required steps, scoring risks, and targeted follow-up drills. This makes the output composable for agents that plan study schedules, run tutoring sessions, issue proof-of-progress credentials, or trigger learning bounties.

Phase 1 deliverable:

- `SKILL.md` with callable actions and workflow.
- JSON input/output schemas.
- Demo transcript.
- Local package validator.
- Submission and demo-video script.

## Pharos Message

Pharos is building the chain for agents. Final Exam Review Coach contributes a reusable education module for that economy: an agent-callable Skill that transforms private course corpora into structured tutoring actions, then hands clean outputs to payment, credential, or on-chain workflow Skills. It is safe by default, composable by design, and useful immediately for students.
